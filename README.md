# ⚡ JAMPAN XMD — ROG AURA BOT

🔥 A modern WhatsApp bot system powered by Baileys with ROG-inspired neon UI, pairing system, and API backend.

---

## 🌌 FEATURES

- 🤖 WhatsApp Pairing System (8-digit code)
- ⚡ Express API backend
- 📊 Bot status checker
- 🔁 Auto reconnect system
- 🧠 Session management
- 🌐 Frontend pairing UI (ROG Aura Design)
- 📱 Mobile responsive web UI
- 🔥 Glassmorphism + RGB animation UI
- 👤 Profile system (future upgrade)
- ⚙️ Command system ready structure
